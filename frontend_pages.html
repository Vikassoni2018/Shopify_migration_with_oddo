Odoo to Shopify Migration

This app helps you migrate Odoo order data into Shopify with preview, conversion, and import flow.

App functionality

  • Upload Odoo CSV and preview converted rows.
  • Generate Matrixify compatible CSV files.
  • Run Shopify order create/update import flow.

How to create Shopify token

  1. Open Shopify Admin → Apps and sales channels.
  2. Create/select a custom app.
  3. Grant Admin API scopes: read_orders, write_orders.
  4. Install app and copy Admin API access token.

How to set domain (with visualization)

Use your store domain in this format: your-store.myshopify.com. Example URL visualization: https://your-store.myshopify.com/admin.
