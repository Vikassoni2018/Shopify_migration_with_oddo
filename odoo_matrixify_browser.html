Local Shopify Sync

Upload Odoo CSV. Connect Shopify. Create missing orders and update existing ones.

This local app keeps your Admin API token on your machine, prepares the CSV for Matrixify, and can also push orders directly into Shopify. Existing orders are updated with Shopify's safe order update fields instead of creating duplicates.

Local token handling The access token is sent only to the localhost app, not to a hosted website.
Create or update Missing orders are created. Existing orders are updated with note, phone, delivery address, and custom attributes.
Matrixify still included You can still download Orders.csv and the mapping report from the same page.
